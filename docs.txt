//Text to speech api

    It is post request and can call with
        http://localhost:8082/text-to-speech
    Body of request contains json formated text

    Input Sample
        {
            "text":"Helly"
        }
    Output Sample
        {
            "audioContent":{
                "type":"Buffer",
                "data":[
                    255,
                    289,
                    ....
                ]
            }
        }
